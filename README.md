# m-web-notes

## 项目简介

`m-web-notes` 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面可能涵盖技术笔记、工具页面、演示示例或各类文档整理，它们彼此独立，不针对任何特定域名或网站。本仓库旨在提供一个干净、可访问的存储空间，方便分享和查看这些页面内容。

## 目录说明

```
m-web-notes/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（如样式、脚本、图片等）
├── index.html      # 仓库首页导航（可选）
└── README.md       # 本文件
```

- **pages/**：每个 HTML 文件代表一个独立的页面，命名清晰，便于识别。
- **assets/**：资源文件按页面或类型组织，避免冲突。

## 页面归档说明

- 所有页面均为静态 HTML，可直接通过浏览器打开。
- 页面之间无依赖关系，可单独使用。
- 部分页面可能包含外部资源引用（如 CDN 库），请确保网络连接正常。
- 页面内容以实用、简洁为主，不包含商业推广或外部链接。
